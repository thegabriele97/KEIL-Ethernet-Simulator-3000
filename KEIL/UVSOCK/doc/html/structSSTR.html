Description

String data

Transfers string data between the client and uVision.

NOTE: In some cases, the string may be longer the the statically defined szStr string length.

This structure is used in the following messages:

Client ==> uVision (Request)

  • UV_PRJ_ENUM_FILES (Set nLen to the length of the NULL terminated string in szStr)

Client <== uVision (Async)

  • UV_PRJ_ENUM_GROUPS_ENU (All members are written by uVision)
  • UV_PRJ_ENUM_FILES_ENU (All members are written by uVision)
  • UV_PRJ_ENUM_TARGETS_ENU (All members are written by uVision)
  • UV_PRJ_GET_CUR_TARGET (All members are written by uVision)*
  • UV_PRJ_SET_OUTPUTNAME (All members are written by uVision)
  • UV_PRJ_GET_OUTPUTNAME (All members are written by uVision)
  • UV_DBG_CMD_OUTPUT (All members are written by uVision)
  • UV_DBG_EVTR_ENUMSCVDFILES_ENU (All members are written by uVision)
