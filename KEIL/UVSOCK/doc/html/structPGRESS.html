Description

uVision Progress bar control data

Indicates how the uVision progress bar should be updated

This structure is used in the following messages:

Client ==> uVision (Request)

  • UV_PRJ_CMD_PROGRESS (Set job to the type of progress bar operation to perform, perc to the percentage complete (UV_PROGRESS_SETPOS only), and szLabel to the progress bar label (UV_PROGRESS_INIT / UV_PROGRESS_INITTXT / UV_PROGRESS_SETTEXT only))
